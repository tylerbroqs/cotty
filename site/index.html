COTTY
Features Quick start GitHub

The multiplayer terminal.

Host your shell. Teammates join from a terminal or a browser — watch together, and type together when you let them. End-to-end encrypted, one binary, open source.

Get Cotty on GitHub Quick start
$ go install github.com/tylerbroqs/cotty/cmd/cotty@latest
$ cotty host --relay relay.example.com:7374
cotty: hosting /bin/zsh via relay relay.example.com:7374 (end-to-end encrypted)
cotty: session code XJ4K2P
cotty: guests join with: cotty join "ws://relay.example.com:7374/ws?code=XJ4K2P#k=8D0Uy…"
cotty: or in a browser: http://relay.example.com:7374/join#code=XJ4K2P&k=8D0Uy…

cotty: alice joined (1 connected)
$ cotty ctl allow alice
alice can now type

Built for working together in a shell

Not screen sharing. A real shared session with permissions, presence, and encryption — the things pair debugging and incident response actually need.

##Multiplayer, not a movie

Guests connect to the live PTY. When you grant them write access, their keystrokes land in the same shell — no lag-smeared video of your terminal.

##End-to-end encrypted

Relayed sessions are sealed with AES-256-GCM by default. The key travels in the URL fragment, which never touches the wire — the relay forwards ciphertext it cannot read.

##Works behind NAT

The host dials out to a relay you can self-host with one command. No port forwarding, no firewall tickets, no agent daemons.

##Join from a browser

Every relay serves an embedded xterm.js client — decryption included, via WebCrypto. Send a link; teammates need nothing installed.

##Per-guest permissions

Everyone joins view-only with a name. cotty ctl allow alice, deny, kick — live, from any terminal on the host, even inside the session itself.

##One small binary

Host, client, relay, and web UI in a single Go binary with no runtime dependencies. MIT licensed, no telemetry, no accounts.

Quick start

Share a shell on your LAN in ten seconds, or across the internet with a relay.

# install
go install github.com/tylerbroqs/cotty/cmd/cotty@latest

# share your shell (guests are view-only by default)
cotty host

# a teammate joins from their terminal…
cotty join -name alice "ws://192.168.1.10:7373/ws?code=XJ4K2P"

# …or from a browser, no install needed
http://192.168.1.10:7373/join#code=XJ4K2P

# let alice type; kick bob
cotty ctl allow alice
cotty ctl kick bob
